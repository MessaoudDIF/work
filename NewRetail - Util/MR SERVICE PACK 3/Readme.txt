Main Retail Update Pack.

Download
--------------------
http://www.mainretail.com/download/updatepack.exe


Install
--------------------
Important: CLOSE ALL MAIN RETAIL SOFTWARE BEFORE RUN THE UPDATE.
YOU MUST RUN THE UPDATE ON THE SERVER INSTALLATION FIRST.

1 - Locate and click on updatepack.exe
2 - The Unzip file will open
3 - Select a folder to extract the update files
4 - Open the folder you extracted the files
5 - Click on MRUpdate.exe
6 - The update pack will open
7 - Click on Start and wait until the update is complete

RUN THE UPDATE PACK ON EACH COMPUTER THAT YOU HAVE INSTALLED MAIN RETAIL.
